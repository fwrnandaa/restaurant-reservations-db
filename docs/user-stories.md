# Historias de Usuario


Como cliente, quiero realizar una reservación en el restaurante para asegurar una mesa en la fecha y hora deseada.

Como empleado del restaurante, quiero registrar los datos de un cliente para guardar su información en el sistema.

Como empleado del restaurante, quiero crear una reservación para un cliente para asegurar su lugar en el restaurante.

Como empleado del restaurante, quiero consultar las reservaciones registradas para saber qué mesas estarán ocupadas.

Como empleado del restaurante, quiero modificar una reservación para actualizar la fecha, hora o número de personas.

Como empleado del restaurante, quiero cancelar una reservación para liberar la mesa para otros clientes.

Como administrador, quiero registrar las mesas del restaurante para llevar control de su capacidad.

Como empleado del restaurante, quiero asignar una mesa a una reservación para organizar a los clientes cuando lleguen.

Como empleado del restaurante, quiero consultar la disponibilidad de mesas para saber si es posible aceptar una reservación.

Como sistema, quiero almacenar la fecha y hora de cada reservación para mantener organizado el registro de reservaciones.
